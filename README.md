# Family Feud Game

ทำเล่นกันสำหรับ Friday Sharing

# Run the Game

1. เปิด backend service

```
node backend/server.js
```

2. เปิด terminal ใหม่และ เปิด frontend service ด้วยคำสั่ง

```
npm run dev
```

3. เปิด browser http://localhost:5173/ กดเปิด display และ host เป็นแท็บใหม่

เลือกคำถามจากหน้า host ถ้าตอบถูกกด reveal ถ้าตอบผิด กด strike ตอบผิด 3 ครั้งจะเปลี่ยนทีม

sound effect https://www.myinstants.com/en/search/?name=family%20feud
